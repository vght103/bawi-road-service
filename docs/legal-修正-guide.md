**이유:** 수속료와 학비의 정의를 명확히 분리해야 환불규정의 법적 근거가 됨

### 1-2. 제6조 (회원 가입) — 미성년자 조항 수정

**위치:** 제6조 2항의 가입 거부 사유 목록

**현재:**

```
만 14세 미만인 경우
```

**변경:**

```
만 18세 미만인 경우 (단, 법정대리인의 동의를 받은 경우는 예외로 하되, 만 14세 미만은 가입 불가)
```

**이유:**
- 성인만 타겟이지만, 어학연수 특성상 만 16~17세 고등학생이 올 수 있음
- 민법상 미성년자(만 19세 미만)의 법률행위는 법정대리인 동의 필요
- 아예 막으면 비즈니스 손실, 동의 절차를 넣으면 법적 안전 + 고객 확보

### 1-3. [신설] 제6조의2 (미성년자 이용)

**위치:** 제6조와 제7조 사이에 신설

**추가 내용:**

```
제6조의2 (미성년자 이용)

1. 만 14세 이상 만 18세 미만의 미성년자가 서비스를 이용하고자 하는 경우, 법정대리인(부모 등)의 동의가 있어야 합니다.

2. 미성년자의 수속 신청은 법정대리인이 다음 사항에 동의한 경우에만 처리됩니다.
   - 이용약관 동의
   - 개인정보 수집·이용 동의 (여권 등 고유식별정보 포함)
   - 수속료 납부 및 환불규정 동의
   - 개인정보 국외이전 동의

3. 회사는 법정대리인의 동의 여부를 이메일 또는 전자서명 등의 방법으로 확인합니다.

4. 법정대리인은 미성년 이용자의 개인정보에 대한 열람, 정정, 삭제, 처리정지를 요청할 수 있습니다.
```

**이유:**
- 개인정보보호법 제22조: 만 14세 미만 아동의 개인정보 수집 시 법정대리인 동의 필수
- 민법 제5조: 미성년자의 법률행위에 법정대리인 동의 필요
- 수속 신청(계약 체결)은 법률행위에 해당

**구현 위치:**
- 회원가입 시: 생년월일 기반으로 미성년자 여부 판별 → 미성년자면 법정대리인 이메일 입력 + 동의 요청 이메일 발송
- 수속 신청 시: 미성년자인 경우 법정대리인 동의 확인 후 진행

### 1-5. [신설] 제10조의2 (수속료 및 결제)

**위치:** 제10조(책임 제한 및 면책) 뒤에 신설

**추가 내용:**

```
제10조의2 (수속료 및 결제)

1. 이용자가 수속을 신청하는 경우, 수속료 100,000원(부가세 포함)을 납부하여야 합니다.

2. 수속료는 어학원 등록 확인, 기숙사 선점, 입학허가서 발급 요청 등 수속 진행을 위한 실비성 비용이며, 수속 신청 접수 즉시 해당 업무가 개시됩니다.

3. 학비(수업료 및 기숙사비)는 이용자가 어학원에 직접 납부합니다. 회사는 학비를 수취하거나 경유하지 않으며, 학비 결제와 관련한 분쟁에 대해 책임을 지지 않습니다.

4. 회사는 이용자와 어학원 간 학비 관련 소통을 지원할 수 있으나, 학비의 환불 여부 및 조건은 해당 어학원의 규정에 따릅니다.
```

**이유:**
- 수속료의 법적 성격을 "실비성 비용"으로 명확히 규정 → 환불 불가의 근거 마련
- 학비가 바위로드를 경유하지 않는 구조를 명시 → 법적 리스크 차단
- 전자상거래법상 가격, 결제 조건 등을 사전 고지해야 함

### 1-6. [신설] 제10조의3 (수속료 환불)

**위치:** 제10조의2 바로 뒤에 신설

**추가 내용:**

```
제10조의3 (수속료 환불)

1. 수속료는 어학원 등록 확인 및 기숙사 선점 등 실비성 비용의 성격으로, 수속 업무가 개시된 이후에는 원칙적으로 환불되지 않습니다.

2. 다만, 다음 각 호의 경우에는 아래 기준에 따라 환불합니다.

   가. 수속 신청 후 회사의 수속 업무 개시 전(어학원 연락 전) 취소를 요청한 경우: 수속료 전액 환불
   나. 회사의 귀책사유(수속 지연, 잘못된 정보 제공 등)로 수속이 정상 진행되지 못한 경우: 수속료 전액 환불
   다. 어학원의 사정(폐원, 입학 불가 등)으로 수속이 진행 불가한 경우: 수속료 전액 환불
   라. 이용자의 단순 변심에 의한 취소: 수속 업무 개시 후에는 환불 불가

3. 환불은 요청일로부터 3영업일 이내에 원래 결제 수단으로 처리합니다.

4. 학비 환불에 관한 사항은 해당 어학원의 환불규정에 따르며, 회사는 이에 대한 책임을 지지 않습니다. 다만, 회사는 이용자의 요청이 있는 경우 어학원과의 소통을 지원할 수 있습니다.
```

**이유:**

harry가 원하는 "수속료 환불불가" 방침을 법적으로 안전하게 만들기 위한 핵심 포인트:

- **"어떤 경우에도 환불 불가"는 위험** → 전자상거래법 제17조(청약철회권)과 충돌
- **"수속 업무 개시 후 환불 불가"는 안전** → 용역이 이미 시작된 경우 청약철회 제한 가능 (전자상거래법 제17조 제2항 제5호)
- 핵심 전략: 수속료를 "선불 서비스 비용(실비)"으로 규정 + "수속 신청 즉시 업무 개시"를 명시하면, 사실상 환불불가와 동일한 효과
- 단, 업무 개시 전 취소와 회사/어학원 귀책 취소는 환불해야 법적으로 안전함

| **회원가입 UI** | 생년월일 입력 → 만 18세 미만 시 법정대리인 동의 플로우 |
| **수속 신청 UI** | 미성년자인 경우 법정대리인 동의 확인 후 진행 |

### 현실적 구현 방안 (1인 유학원)

MVP 단계에서는 시스템 자동화보다 수동 프로세스가 현실적:

1. **회원가입 시**: 생년월일 입력 → 미성년자면 안내 문구 표시
   - "만 18세 미만의 경우, 수속 신청 시 법정대리인 동의가 필요합니다"
2. **수속 신청 시**: 미성년자면 추가 체크박스
   - "법정대리인(부모님)의 동의를 받았습니다"
   - 법정대리인 이메일 입력 → 동의 확인 이메일 발송 (간단한 구글폼 링크로도 가능)
3. **카카오톡 상담에서 직접 확인**: 미성년자라면 상담 중 부모님 동의 여부 확인
