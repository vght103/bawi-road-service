# 어학원 검색 및 상세

> 상태: 구현완료
> 최종 수정일: 2026-03-11
> 관련 페이지: src/pages/academies.astro, src/pages/academy/[id].astro, src/react-pages/academy/

## 개요

어학원 목록을 검색/필터링하여 비교하고, 개별 어학원의 상세 정보(기본 정보, 장단점, 코스, 기숙사, 가격)를 확인할 수 있는 기능.

## 문제 정의

필리핀 어학연수를 고려하는 사용자가 여러 어학원을 조건별로 비교하고, 각 어학원의 상세 정보를 한 곳에서 확인할 수 있어야 한다.

## 사용자 스토리

- [x] 사용자는 어학원 목록을 텍스트 검색으로 찾을 수 있다
- [x] 사용자는 지역(세부/바기오)으로 어학원을 필터링할 수 있다
- [x] 사용자는 코스 태그(스파르타, ESL, IELTS 등)로 어학원을 필터링할 수 있다
- [x] 사용자는 어학원 카드를 클릭하여 상세 페이지로 이동할 수 있다
- [x] 사용자는 어학원 상세 페이지에서 기본 정보, 장단점, 코스, 기숙사 정보를 확인할 수 있다
- [x] 사용자는 상세 페이지에서 견적 요청 또는 수속 신청으로 이동할 수 있다

## 요구사항

### Must (필수)

- REQ-007: 텍스트 검색으로 어학원명, 지역, 설명을 검색할 수 있다
- REQ-008: 지역 필터(전체/세부/바기오)로 어학원을 필터링할 수 있다 (단일 선택)
- REQ-009: 코스 필터(스파르타, ESL, IELTS 등)로 어학원을 필터링할 수 있다 (복수 선택, OR 조건)
- REQ-010: 어학원 카드에 대표 이미지, 이름, 한 줄 설명, 지역/수업방식 뱃지, 코스 태그를 표시한다
- REQ-011: 어학원 상세 페이지에 이미지 슬라이더, 기본 정보, 장점/참고사항, 추천 대상, 소개, 시설, 코스, 기숙사를 표시한다
- REQ-012: 상세 페이지에 무료 견적/수속 신청/카카오톡 상담 CTA를 제공한다

## 상세 스펙

### UI/UX

#### 검색 페이지 (academies)

| 영역 | 구성 |
|------|------|
| 헤더 | "어학원 비교" 타이틀 |
| 검색/필터 바 | 텍스트 검색 + 지역 필터 + 코스 필터 (sticky, top-16 고정) |
| 결과 | "{N}개 어학원" 카운트 + 카드 그리드 (1/2/3열 반응형) |

- 필터: 텍스트 AND 지역 AND 코스(OR) 조합
- 검색 결과 0건: "검색 결과가 없습니다." 표시
- 전체 데이터 0건: "등록된 어학원이 없습니다." 표시

#### 상세 페이지 (academy/[id])

| 순서 | 섹션 | 조건 |
|------|------|------|
| 1 | 브레드크럼 (홈 > 어학원 > 이름) | 항상 |
| 2 | 이미지 슬라이더 | 2장 이상이면 슬라이더, 1장이면 단순 이미지 |
| 3 | 뱃지 + 이름 + 설명 | 항상 (shortDesc 우선, 없으면 desc) |
| 4 | 기본 정보 카드 | 값이 있는 항목만 표시 |
| 5 | 장점 / 참고사항 | 데이터 있을 때만 |
| 6 | 추천 대상 | 데이터 있을 때만 |
| 7 | 어학원 소개 | 데이터 있을 때만 |
| 8 | 시설 | 데이터 있을 때만 |
| 9 | 코스 안내 | 항상 (1:1/그룹/선택 시간 표시) |
| 10 | 기숙사 | 항상 (타입별 주당 가격, 만원 단위) |
| 11 | CTA 버튼 | 견적 / 수속 / 카카오톡 |

### 데이터 모델

- `academies` 테이블 — 목록 조회 (검색 페이지)
- `academies` 테이블 — 단건 조회 by id (상세 페이지)
- 필드: name, region, desc, shortDesc, images[], tags[], academy_system, established_year, capacity, korean_ratio, location_detail, address, website, pros[], cons[], recommendedFor[], description, facilities[], courses[], dormitories[]

### API

- `fetchAcademies()` — 전체 어학원 목록 조회
- `fetchAcademy(id)` — 단건 어학원 상세 조회

### 비즈니스 로직

- 텍스트 검색: name(소문자 비교), region, desc에서 부분 일치
- 지역 필터: 단일 선택, "전체" = null (필터 해제)
- 코스 필터: 복수 선택, academy_system + tags 중 하나라도 일치하면 표시 (OR)
- 기숙사 가격: pricePerWeek(원) → 만원 단위로 변환하여 표시
- 코스 선택 수업: 0시간이면 숨김

### 엣지케이스

- 어학원 로딩 중: LoadingOverlay 표시
- 존재하지 않는 academy id: "어학원을 찾을 수 없습니다." + 목록으로 돌아가기 링크
- 이미지 1장: 슬라이더 대신 단순 이미지
- 선택 필드(pros, cons, facilities 등) 없음: 해당 섹션 미표시
