# 홈 페이지

> 상태: 구현완료
> 최종 수정일: 2026-03-11
> 관련 페이지: src/pages/index.astro, src/react-pages/home/

## 개요

바위로드 서비스의 랜딩 페이지. 서비스 핵심 가치(수수료 0원, 가격 공개)를 전달하고, 주요 기능(AI 상담, 견적 요청, 어학원 탐색, 1:1 상담)으로의 진입점을 제공한다.

## 문제 정의

필리핀 어학연수를 알아보는 사용자가 처음 접하는 페이지로, 서비스의 신뢰성과 차별점을 빠르게 전달하고 다음 행동(견적 요청, AI 상담, 어학원 탐색)으로 유도해야 한다.

## 사용자 스토리

- [x] 사용자는 바위로드가 어떤 서비스인지 한눈에 파악할 수 있다
- [x] 사용자는 AI 상담 입력창에 질문을 입력하여 채팅 페이지로 이동할 수 있다
- [x] 사용자는 무료 견적 카드를 통해 견적 요청 페이지로 이동할 수 있다
- [x] 사용자는 인기 어학원 슬라이더를 통해 어학원을 탐색할 수 있다
- [x] 사용자는 수속 절차 4단계를 확인할 수 있다
- [x] 사용자는 1:1 상담 신청 버튼을 통해 문의 페이지로 이동할 수 있다

## 요구사항

### Must (필수)

- REQ-001: 히어로 섹션에 서비스 핵심 문구와 AI 상담 입력창을 표시한다
- REQ-002: 무료 견적 카드에 핵심 포인트 3가지를 표시하고, 견적 페이지로의 CTA 버튼을 제공한다
- REQ-003: 인기 어학원을 Swiper 슬라이더로 최대 9개 표시한다 (모바일 1개, 태블릿 2개, 데스크탑 3개씩)
- REQ-004: 바위로드 차별점 3가지(가격 공개, 솔직 비교, 수수료 제로)를 카드로 표시한다
- REQ-005: 수속 절차 4단계(무료상담→어학원확정→수속→출국)를 표시한다
- REQ-006: CTA 섹션에서 바위로드 서포트 소개 + 상담 신청 버튼을 제공한다

## 상세 스펙

### UI/UX

#### 섹션 구성 (위에서 아래 순서)

| # | 섹션 | 구성 요소 |
|---|------|----------|
| 1 | 히어로 | 좌: 메인 문구 + "수수료 0원" 뱃지 + AI 상담 입력 카드 / 우: 무료 견적 카드 |
| 2 | 신뢰 지표 바 | "수수료 0원", "가격 100% 공개", "검증된 어학원만", "장단점 솔직 비교" |
| 3 | 인기 어학원 | Swiper 슬라이더 + "전체 어학원 보기" 링크 |
| 4 | 차별점 | 카드 3개 (가격 공개 / 솔직 비교 / 수수료 제로) |
| 5 | 수속 절차 | 4단계 카드 + "자세한 절차 보기" 링크 |
| 6 | CTA | 서포트 소개 + "자세히 알아보기" + "1:1 상담 신청" 버튼 |

#### 반응형

- 모바일: 1열 레이아웃, 슬라이더 1개씩
- 태블릿(768px+): 슬라이더 2개씩
- 데스크탑(1024px+): 2열 히어로, 슬라이더 3개씩, 3열 차별점/4열 절차

#### 인터랙션

- 스크롤 시 `.reveal` 요소 페이드인 애니메이션 (IntersectionObserver, threshold 0.1)
- AI 상담 입력: Enter 키 또는 전송 버튼 클릭 → `/chat?q={키워드}`로 이동
- 어학원 슬라이더: loop 모드, 데스크탑에서 좌우 화살표 버튼 표시

### 데이터 모델

- `academies` 테이블에서 전체 어학원 목록 조회 (useQuery)
- 슬라이더에는 최대 9개만 표시 (`.slice(0, 9)`)

### API

- `fetchAcademies()` — Supabase에서 어학원 목록 조회

### 비즈니스 로직

- AI 상담 입력값이 비어있으면 전송 버튼 비활성화
- 어학원 카드에 지역(region)과 수업 방식(academy_system) 뱃지 표시
- 이미지는 `loading="lazy"` 적용

### 엣지케이스

- 어학원 데이터 로딩 중: "로딩 중..." 텍스트 표시 (Suspense fallback)
- 어학원 데이터 0건: 빈 슬라이더 (별도 빈 상태 UI 없음)
- AI 상담 빈 입력 제출: 이동하지 않음 (trim 후 빈 값 체크)
