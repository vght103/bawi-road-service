# 정보 페이지 (연수 절차 / 비자 안내 / 왜 필리핀)

> 상태: 구현완료
> 최종 수정일: 2026-03-11
> 관련 페이지: src/pages/process.astro, src/pages/visa-info.astro, src/pages/why-philippines.astro

## 개요

필리핀 어학연수 관련 정보를 제공하는 3개의 정적 페이지. React 컴포넌트나 API 호출 없이 순수 Astro HTML + Tailwind로 구성된다.

## 문제 정의

사용자가 어학연수를 결정하기 전에 필요한 정보(절차, 비용, 장점)를 쉽게 확인할 수 있어야 한다. 별도 상담 없이도 기본 정보를 스스로 파악할 수 있는 콘텐츠가 필요하다.

## 사용자 스토리

- [x] 사용자는 어학연수 절차와 준비물을 단계별로 확인할 수 있다
- [x] 사용자는 비자 종류와 현지 비용을 확인할 수 있다
- [x] 사용자는 필리핀 어학연수의 장점을 비교하여 확인할 수 있다

## 상세 스펙

### 연수 절차 & 준비물 (`/process`)

#### 절차 타임라인 (6단계)

1. 상담 및 어학원 선정
2. 등록 및 비용 납부
3. 입학허가서 수령
4. 여권 및 항공권 준비
5. 보험 및 e-Travel 등록
6. 출국

- 각 단계마다 TIP 콜아웃 박스 포함

#### 준비물 체크리스트 (3개 카드)

- 성인 (만 15세 이상): 여권, 항공권, 여행자보험, 증명사진 등
- 가족연수: 가족관계증명서 추가
- 주니어 단독 (만 15세 미만): WEG 서류, 공증 위임장 등 추가

#### CTA

- 어학원 검색 → `/academies`
- 비자 안내 → `/visa-info`

### 비자 & 현지비용 안내 (`/visa-info`)

#### 스티키 앵커 네비게이션

SSP | 비자 연장 | ACR-I Card | 기타 현지비용

#### SSP (Special Study Permit)

- 6,500페소 (~15만원), 6개월 유효, 어학원에서 대행

#### 비자 연장 비용표

| 기간 | 비용 |
|------|------|
| ~30일 | 무비자 (무료) |
| ~59일 | 약 4,140페소 |
| ~89일 | 약 5,410페소 |
| ~119일 | 약 3,440페소 |
| ~149일 | 약 3,440페소 |
| ~179일 | 약 3,440페소 |

- TIP: 대사관에서 59일 비자 사전 발급 가능

#### ACR-I Card

- 3,500페소, 59일 이상(8주+) 체류 시 필수

#### 기타 현지비용 요약

- 4주 총 예상: ~20~25만원
- 12주 총 예상: ~55~70만원

#### CTA

- 무료 견적 → `/quote?from=visa-info`
- 카카오톡 상담

### 왜 필리핀 어학연수인가요 (`/why-philippines`)

#### 스티키 앵커 탭 네비게이션

1:1 수업 | 스파르타 시스템 | 기숙형 어학원 | 합리적 비용 | 지리적 장점

#### POINT 01 — 1:1 맨투맨 수업

- 하루 4~6시간 개인 맞춤 커리큘럼

#### POINT 02 — 학습 스타일 비교 (3컬럼)

- 스파르타: 10~12시간/일, 평일 외출 금지
- 세미스파르타: 8~10시간/일, 저녁 외출 가능
- 자율형: 6~8시간/일, 자유로운 일정

#### POINT 03 — 기숙형 올인원 시스템

- 숙소/식사/수업 원캠퍼스, 3식 제공, 세탁, 한국인 매니저

#### POINT 04 — 비용 비교 (3컬럼, 다크 배경)

- 필리핀: ~280만원 (8주, 숙식 포함)
- 호주/캐나다: ~600~800만원 (학비만)
- 미국/영국: ~800~1,200만원 (학비만)

#### POINT 05 — 지리적 장점 (3카드)

- 직항 4시간
- 시차 1시간
- 다양한 액티비티

#### CTA

- 어학원 검색 → `/academies`
- 연수 절차 확인 → `/process`
