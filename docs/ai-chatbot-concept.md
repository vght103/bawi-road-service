# AI 상담 챗봇 컨셉 문서

> **작성일**: 2026-02-20
> **상태**: 컨셉 확정, 구현 전

---

## 개요

홈 히어로 섹션의 정적 챗봇 목업을 **실제 AI 챗봇**으로 전환한다.
홈에서는 진입점 역할만 하고, 본격적인 대화는 **별도 채팅 페이지(`/chat`)**에서 진행한다.

---

## 사용자 플로우

```
홈 히어로 챗봇 카드 (input 활성화)
  ↓ 메시지 입력 + 전송 클릭
/chat?q={입력내용} 페이지로 이동
  ↓ 첫 메시지 자동 전송
AI 응답 (텍스트 / 어학원 카드 / CTA 버튼)
```

### 홈 히어로 변경사항

- [x] 플로팅 뱃지 2개 제거 ("1:1 수업 하루 4시간", "세부 직항 4시간")
- [x] input `disabled` 해제 → 활성화
- [x] 전송 버튼 클릭 시 `/chat?q={입력내용}` 으로 라우팅
- [x] 기존 하드코딩된 대화 내용은 유지 (미리보기 역할)

## 채팅 페이지 (`/chat`)

### UI 구조

```
┌─────────────────────────────────┐
│  ←  AI 상담 어시스턴트            │  ← 헤더 (뒤로가기 + 타이틀)
├─────────────────────────────────┤
│                                 │
│  🤖 AI                          │
│  ┌─────────────────────┐       │  ← AI 메시지 (왼쪽, 아바타)
│  │ 안녕하세요! 필리핀     │       │
│  │ 어학연수 궁금한 점을   │       │
│  │ 물어보세요.           │       │
│  └─────────────────────┘       │
│                                 │
│       ┌──────────────────┐     │
│       │ 세부 스파르타      │     │  ← 유저 메시지 (오른쪽)
│       │ 어학원 추천해주세요 │     │
│       └──────────────────┘     │
│                                 │
│  🤖 AI                          │
│  ┌─────────────────────┐       │
│  │ 세부 스파르타 어학원을  │       │
│  │ 추천드릴게요!         │       │
│  └─────────────────────┘       │
│  ┌───────┐ ┌───────┐          │  ← 어학원 카드 목록 (가로 스크롤)
│  │ SMEAG │ │ EV    │          │
│  └───────┘ └───────┘          │
│                                 │
├─────────────────────────────────┤
│  [어학연수에 대해 물어보세요...] 📤│  ← 고정 input bar
└─────────────────────────────────┘
```

### AI 응답 타입

| 사용자 질문 유형 | AI 응답 형태 |
|-----------------|-------------|
| 일반 상담 질문 | 텍스트 응답 |
| 어학원 추천 요청 | 텍스트 + **어학원 카드 목록** |
| 비용/가격 관련 질문 | 텍스트 + **"견적 받기" CTA 버튼** (→ `/quote`) |
| 비자/준비 과정 질문 | 텍스트 응답 (관련 페이지 링크 포함) |

### 어학원 카드 (채팅 내 노출)

```
┌──────────────────┐
│  [어학원 이미지]    │
│  SMEAG Capital    │
│  세부 · 스파르타    │
│  [자세히 보기 →]   │  ← 클릭 시 /academy/{id} 이동
└──────────────────┘
```

### CTA 버튼 (채팅 내 노출)

비용 관련 질문 시:
```
┌────────────────────────┐
│  📋 무료 견적 받기 →     │  ← 클릭 시 /quote 이동
└────────────────────────┘
```

## AI 동작 방식

### 시스템 프롬프트 (개요)

- 바위로드 AI 상담 어시스턴트 역할
- 필리핀 어학연수 전문 상담
- 어학원 DB 정보 기반 응답
- **비용 계산은 하지 않음** → 견적 페이지로 유도
- 어학원 추천 시 DB에서 조건에 맞는 어학원 조회 후 구조화된 데이터로 반환

### 응답 포맷 (AI → 프론트엔드)

```json
{
  "message": "세부 지역 스파르타 어학원을 추천드릴게요!",
  "components": [
    {
      "type": "academy_cards",
      "data": [
        { "id": "smeag", "name": "SMEAG Capital", "region": "세부", "style": "스파르타" }
      ]
    }
  ]
}
```

```json
{
  "message": "정확한 비용은 기간, 코스, 기숙사 옵션에 따라 달라져요. 무료 견적을 받아보시는 게 가장 정확합니다!",
  "components": [
    {
      "type": "cta_button",
      "data": { "label": "무료 견적 받기", "link": "/quote" }
    }
  ]
}
```
